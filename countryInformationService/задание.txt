Написать сервис, который по фильтру будет отдавать информацию по странам. При инициализации приложения данные загружаются из стороннего сервиса
(link на json : https://restcountries.eu/rest/v2/all) и сохраняются в локальную реляционную БД ( к примеру h2 ).

Полученные из стороннего сервиса данные необходимо десериализовать в модели и сохранить с использованием ORM в БД.

Разрабатываемый сервис должен уметь :

1.     отдавать данные по странам, применяя к ним фильтр:

a.     по имени (регистронезависимый поиск)

b.     по домену (выбор по вхождению)

2.     удалять все данные из локальной БД и наполнять заново из удаленного сервиса


Необязательно сохранять все поля информации по стране из json стороннего сервиса, достаточно :

1.     name

2.     topLevelDomain

3.     population

4.     latlng

5.     currencies

6.     languages

7.     translations

8.     flag

9.     regionalBlocs

Версия jdk : Java 8

Желательный стэк:

1.     Spring ( можно Boot)

2.     Hibernate

3.     maven/gradle

Плюсом будет покрытие кода unit-тестами.

Задание необходимо залить на github и предоставить ссылку на репозиторий.
